Getting Started¶

This getting started guide aims to get you using SISPPEO productively as quickly as possible. It is designed as an entry point for new users, and it provides an introduction to SISPPEO’s main concepts.

Installation¶

Working with conda?

SISPPEO can be downloaded and installed using conda-forge:

conda install sisppeo -c conda-forge
